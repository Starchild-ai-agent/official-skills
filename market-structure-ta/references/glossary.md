# PA / SMC / ICT / Wyckoff / Brooks Glossary

Quick lookup. Sorted alphabetically.

| Term | Meaning |
|---|---|
| **AMD** | Accumulation, Manipulation, Distribution — ICT's PO3 model |
| **AR** | Automatic Rally (Wyckoff accumulation) / Automatic Reaction (distribution) |
| **BC** | Buying Climax — Wyckoff distribution top |
| **Breaker Block** | Failed order block that flips role after price breaks through it |
| **BOS** | Break of Structure — break of a recent swing high (bullish) or low (bearish), continues existing trend |
| **BSL** | Buy-Side Liquidity — stops above prior highs (long stops) |
| **CHoCH** | Change of Character — break of a swing point that reverses prior structure direction; first sign of trend change |
| **CRT** | Candle Range Theory (Romeo Bot's PO3 variant) |
| **EQH** | Equal Highs — magnet for buy-side liquidity |
| **EQL** | Equal Lows — magnet for sell-side liquidity |
| **FVG** | Fair Value Gap — 3-candle imbalance where price moved too fast; gap between candle 1 wick and candle 3 wick |
| **H1, H2, H3** | Brooks pullback counting in bull trends |
| **HH** | Higher High |
| **HL** | Higher Low |
| **IFVG** | Inversion FVG — FVG that got violated and now reverses role |
| **Imbalance** | Same as FVG; area of inefficient price delivery |
| **IPDA** | Interbank Price Delivery Algorithm — ICT's term for institutional price engine |
| **Killzone** | High-probability trading window (Asian, London, NY) |
| **LH** | Lower High |
| **Liquidity** | Where stops are stacked = where price wants to go |
| **Liquidity Sweep** | Brief move that takes out stops then reverses (stop hunt) |
| **Liquidity Void** | Gap-like inefficient move, likely to be filled |
| **LL** | Lower Low |
| **LPS** | Last Point of Support (Wyckoff Phase D in accumulation) |
| **LPSY** | Last Point of Supply (Wyckoff Phase D in distribution) |
| **MSS** | Market Structure Shift — same as CHoCH |
| **OB** | Order Block — last opposite-color candle before strong impulsive move that broke structure |
| **OTE** | Optimal Trade Entry — 0.62–0.79 Fib retracement zone |
| **POI** | Point of Interest — any zone where price might react (OB, FVG, breaker, supply/demand) |
| **PO3** | Power of Three — AMD cycle |
| **PSY** | Preliminary Supply (Wyckoff distribution early signal) |
| **PS** | Preliminary Support (Wyckoff accumulation early signal) |
| **SC** | Selling Climax (Wyckoff accumulation bottom) |
| **SMS** | Sub-Minor Shift (small CHoCH on LTF) |
| **SOS** | Sign of Strength (Wyckoff accumulation phase D rally) |
| **SOW** | Sign of Weakness (Wyckoff distribution phase D drop) |
| **Spring** | Wyckoff false breakdown below TR low; traps shorts, sweeps SSL |
| **SSL** | Sell-Side Liquidity — stops below prior lows (long stops) |
| **ST** | Secondary Test (retest of climax) |
| **TR** | Trading Range (sideways consolidation) |
| **UT** | Upthrust — Wyckoff false breakout above TR high |
| **UTAD** | Upthrust After Distribution — highest-quality UT, often final high |
| **Unicorn** | Breaker + FVG overlap inside HTF OB — A+ setup |
| **Wick Reject** | Long wick beyond a level then close back inside = rejection of that level |

## Acronym Hierarchy Cheat Sheet
- **Structure direction:** BOS = continuation. CHoCH/MSS = reversal.
- **Liquidity:** BSL above, SSL below.
- **POIs:** OB (origin), FVG (imbalance), Breaker (failed OB), Mitigation (failed swing).
- **Wyckoff sweep:** Spring (bullish) = sweep SSL. UTAD (bearish) = sweep BSL.

## Cross-Framework Synonyms
| Wyckoff | SMC | Brooks |
|---|---|---|
| Spring | Liquidity sweep of SSL | Failed breakout / reversal bar at lows |
| UTAD | Liquidity sweep of BSL | Failed breakout / reversal bar at highs |
| SOS | BOS (bullish) | Strong bull breakout bar |
| SOW | BOS (bearish) | Strong bear breakout bar |
| LPS | Pullback to OB / FVG | H2 pullback |
| Phase A (stopping) | First CHoCH | First reversal bar |
| Phase D (markup begins) | After BOS confirmed | Always-in long established |
